# Solvery na OK

## Wymagania
Solvery działają na bibliotekach standardowych Pythona, nie trzeba nic doinstalowywać.
Testowane na wersji Pythona `3.14`

## Używanie

```bash
$ python ./src/main.py
```

Program wyświetla menu z dostępnymi solverami. Można wybrać, który chcemy i następnie podać dane z zadania.

## Disclaimer

Nie daje gwarancji, że solvery są poprawne, 3 różne LLM'y mówią, że są zgodne z wykładem. Osobiście gwarancji nie daje, weryfikujcie rozwiązania. <br>
Ewentualne poprawki/uwagi piszcie do mnie na DM.
