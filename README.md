# Solvery na OK

## Instalacja

Solvery wymagają pakietu NumPy

```sh
pip install numpy
```

## Używanie

W pliku `./src/main.py` macie przykładowe użycie solverów, ewentualnie wrzućcie w chat to wam wyjaśni dokładnie co i jak działa.

## Disclaimer

Nie daje gwarancji, że solvery są poprawne, 3 różne LLM'y mówią, że są zgodne z wykładem. Osobiście gwarancji nie daje, weryfikujcie rozwiązania
